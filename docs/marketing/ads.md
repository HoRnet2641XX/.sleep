# .nemuri — 広告アセットライブラリ

> `.prompts/04-marketing-assets.md` に基づき生成。
> ターゲット: 日本国内の不眠・中途覚醒・早朝覚醒に悩む成人（25–55歳）。
> トーン: 静かで寄り添い型、押し付けない、具体的な体験ベース。

```
7. 薬・寝具・習慣 実体験レポ
8. 中途覚醒に効いたもの探し
9. 早朝覚醒に悩む仲間と繋がる
10. あなたに似た人に効いたもの
11. 無料・広告なし・登録30秒
12. 睡眠の悩みを一人で抱えない
13. 体験ベースの睡眠改善ヒント
14. 眠れない人のためのSNS
15. 今夜から、変わるかもしれない
```

**説明文（4個まで、各90字以内）**

```
1. 不眠・中途覚醒・早朝覚醒に悩む人同士が「自分に効いたもの」を共有するSNS。薬、サプリ、寝具、習慣 — 試した人の本物の体験だけが集まります。
2. あなたの症状に近い人のレビューだけを見つけられます。広告じゃない、本物の声。登録無料、30秒で始められます。
3. 「これが効いた」「全く効かなかった」両方の声が集まるから、失敗しない選び方ができます。同じ悩みの仲間とつながろう。
4. プロフィールに睡眠の悩みを登録すると、あなたに似た人のおすすめだけが届きます。今夜から、ひとりじゃない。
```

### A/B テスト軸

- **A案**: ベネフィット訴求（「効いたものだけ集まる」）
- **B案**: 共感訴求（「眠れない夜をひとりにしない」）
- **C案**: 具体症状訴求（「中途覚醒」「早朝覚醒」）
- → 2週間ごとに CTR/CVR 最下位を差し替え

### CTA

- 主: `無料で始める`
- 副: `レビューを見る`（会員登録なしで閲覧可のLPセクションへ誘導）

## 2. Meta 広告（Facebook / Instagram）

### フォーマット: シングル画像（フィード・ストーリー）

#### A案：共感訴求（深夜タイムライン向け）

**プライマリーテキスト**
```

スマホを見ても、眠れる気がしない。

「自分に効いたもの」を共有するSNSです。

広告じゃない、本物のレビューだけ。
あなたと同じ悩みの人が、
```

**見出し**: 眠れない夜を、ひとりにしない
**説明**: 不眠の体験共有SNS・登録無料
**CTA ボタン**: 登録する

- 絵文字・ストックフォト禁止

## 3. TikTok 広告

### TikTok ならではの方針

- ネイティブ感を重視（広告感を消す）
- 1作目は 6秒・音ありで
- フォント: システム標準でラフに（作り込みすぎない）
- 「これ TikTok で見て」と言われる動画感

---

**シナリオ**
```

テロップ: また眠れない

[0:02] 真っ暗な部屋、天井を見上げるPOV
テロップ: サプリ試した。効かなかった。

[0:04] 本を開くカット
テロップ: 本読んだ。変わらなかった。

[0:06] スマホを取るカット
テロップ: でも、これだけは違った。

[0:08] .nemuri アプリ画面
テロップ: 同じ悩みの人のレビューだけが集まるSNS

[0:11] レビューをスクロールするUI
テロップ: 広告じゃない、本物の体験

[0:13] マスコット表示
```

## 運用上の注意

1. **薬機法**: 「治る」「改善する」「効果がある」など断定表現禁止。常に「体験ベース」の言い回しで
2. **景表法**: 「No.1」「日本初」等の最上級表現は根拠必須
3. **配慮**: 睡眠障害は医療領域に近いため、医療行為の推奨にならないよう注意
4. **メンタルヘルス配慮**: 「死にたくなるほど」等の過激表現を使わない
